Dask (dagster_dask)
-------------------
See also the `Dask deployment guide <https://docs.dagster.io/docs/deploying/dask/>`_.

Python version support
^^^^^^^^^^^^^^^^^^^^^^
Because Dask has dropped support for Python versions < 3.6, we do not test dagster-dask on
Python 2.7 or 3.5.
